Aphro
=====

找到喜欢的电影。



Prerequisites
====

MongoDB


How To Run
===

```
git clone https://github.com/alayii/Aphro.git

cd Aphro && npm install

```

For Dev

```
DEBUG=Aphro ./bin/www
```


Components
===

### 已使用


* Handlebars

* Semantic UI

* Mongoose

* Request

### 预备使用

* Cheerio



路线图
===

1. 网站的基本逻辑。
2. 豆瓣 API npm module.
3. YTS API npm module.
