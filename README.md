Aphro
=====

找到喜欢的电影。



How To Run
===

```
git clone https://github.com/alayii/Aphro.git

cd Aphro && npm install

```

For Dev

```
DEBUG=Aphro ./bin/www
```


Components
===

## 已使用

模板引擎

Handlebars

前端框架

Semantic UI

## 预备使用

MongoDB

Cheerio(用于爬虫)



路线图
===

1. 网站的基本结构。
2. 后端的爬虫程序。
3. 分离爬虫程序，独立为模块。
